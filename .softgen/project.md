# Encuesta de Diagnóstico ERP - Unión de Crédito

## Vision
Sistema de encuesta de diagnóstico para evaluar la madurez operativa y financiera de una unión de crédito. Permite a múltiples roles (contabilidad, tesorería, operaciones, TI) evaluar 7 áreas clave del negocio: gestión de ingresos, control de egresos, conciliación bancaria, flujo de efectivo, cumplimiento fiscal, activos fijos y reporteo financiero. El sistema detecta brechas operativas y genera insights para propuestas de implementación ERP.

Target: Ejecutivos y personal operativo de instituciones financieras que buscan modernizar sus sistemas de control.

## Design
Estilo: Corporativo profesional con claridad accesible (Linear meets financial dashboards)

**Colores (HSL):**
- `--primary: 217 91% 24% (azul corporativo profundo)`
- `--background: 220 14% 96% (gris perla)`
- `--foreground: 222 47% 11% (carbón)`
- `--accent: 142 76% 36% (verde corporativo)`
- `--muted: 220 13% 91% (gris claro)`
- `--card: 0 0% 100% (blanco)`
- `--border: 220 13% 88% (divisor sutil)`

**Tipografía:**
- Headings: Work Sans (600, 700)
- Body: Inter (400, 500, 600)

**Dirección:** Limpio, estructurado, espaciado generoso. Sombras sutiles para elevación. Bordes suaves (rounded-lg). Indicadores de estado claros.

## Features
- Encuesta multi-sección con 7 categorías financieras
- Preguntas cerradas (escala Likert, sí/no) y abiertas
- Opción "No es mi rol" para cada pregunta
- Barra de progreso por sección
- Validación de campos requeridos
- Vista de resumen de respuestas
- Exportación de resultados
- Diseño responsive para desktop y tablet
